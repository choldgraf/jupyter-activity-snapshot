Jupyter Activity Update¶

This book shows recent activity in the Jupyter ecosystem on GitHub.

Each page to the left has a report for one Jupyter GitHub organization. Browse the content to see what Jovyans have been up to!

How this works¶

The data is updated roughly periodically and contains information about the last 60 days of activity, using the github-activity package to scrap the data, Jupyter Book to build a book out of the analyses, and this GitHub Action to scrape the data and build the book before pushing to GitHub Pages.

See the GitHub repository for more code and data!
